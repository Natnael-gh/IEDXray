
# IEDXray
Repository for models and weights trained on the **IEDXray Dataset** — an open-access benchmark for concealed Improvised Explosive Device (IED) detection in X-ray security imaging.

## 📘 Overview
The **IEDXray dataset** was developed to support research in automated explosive threat detection. It includes over **17,360 dual-energy X-ray images** representing realistic concealment scenarios, ranging from isolated threats to highly cluttered baggage scenes.
